# datacleaner
A lightweight Python library for cleaning and standardizing pandas DataFrames. Built for freelance data projects —    handles null filling, type casting, outlier capping, and deduplication via a simple profile-based configuration.        Designed to be imported into client-specific projects rather than used as a one-size-fits-all tool.  

Installation — the pip install -e command                                                                          
  - Quick start — the 5-line usage example showing Cleaner being imported and used                                     
  - Profile structure — a small example of what a profile dict looks like                                              
  - What it does / doesn't do — cleaning only, merging is the caller's responsibility                                  
  - Requirements — Python 3.10+, pandas, numpy, pyarrow
